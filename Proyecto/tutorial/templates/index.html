Pasos para que funcione perfectamente la aplicacion

  1. Configurar POSTGRES_DSN para apuntar a la base de datos de PostgreSQL.
  2. Ejecuta python ./database.py para crear e inicializar la tabla.
  3. Arranca la app: python ./app.py.
  4. Visita http://127.0.0.1:5000/ en el navegador y observa que ya se encuentra funcionando programa integrando las herramientas Flask, PostgreSQL y TailWind sin ningun inconveniente.
